# OM-Tour-and-Travels

Simple static website for OM Tours and Travels, a taxi and tourist service based in Mandi, Himachal Pradesh.

Pages:
- `index.html` — homepage with service overview and media gallery
- `services.html` — service details and route highlights
- `about.html` — brand story and local focus

Assets:
- `assets/van.svg`
- `assets/map.svg`
- `assets/scenery.svg`

Styles:
- `styles.css` — lightweight responsive styling
